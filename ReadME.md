# ByteBites Backend

## Project Overview
ByteBites is a campus food ordering app that aims to make ordering fast, personalized, and smart. This project focuses on the backend logic, handling customers, menu items, and transactions. You can filter and sort menu items, track customer purchases, and calculate order totals.

## TF Summary
**Core Concept:**  
The key idea was learning to break a real-world feature request into classes, attributes, and relationships, and then implement it in Python. Students needed to connect design (UML) with actual code and logic.  

**Common Challenges:**  
Many students would struggle with making sure methods do exactly what they’re supposed to like filtering menus correctly or calculating totals consistently.  

**AI Assistance:**  
AI was really useful for generating initial class structures, UML diagrams, and draft methods.  

**AI Pitfalls:**  
Sometimes AI adds extra features or incorrect logic, so it’s important to verify everything manually.  

**Guidance Without Giving the Answer:**  
A good way to help a student is to have them walk through a small example order by hand, checking what the total should be, before coding it. This helps them reason about the logic without giving away the solution.
